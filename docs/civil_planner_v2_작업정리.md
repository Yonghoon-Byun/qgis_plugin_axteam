## 1. 개요

civil_planner 플러그인을 사용자 피드백 13건을 반영하여 개선한다.
- 위자드 단계: 6단계 → **7단계**로 재구성
- 레이어 구성: 불필요 3종 제거, 행정경계 4분리, 하천 라벨 추가
- 신규 기능: 생성방식 선택, 경과시간 표시, 드래그앤드롭, 인코딩 처리, 템플릿 스타일

## 6. 주요 기술 구현 사항

### R1: 생성방식 선택 (Step 3)

- 라디오 버튼: "임시 (메모리)" / "영구 (파일 저장)"
- 영구 선택 시 `QFileDialog.getExistingDirectory`로 폴더 지정
- `shared_data["layer_save_mode"]` = `"memory"` | `"permanent"`
- `shared_data["save_folder"]` = 경로 | None
- Step 4, 5 모두 이 설정 참조

### R2: 경과시간 표시 (Step 4)

- `QTimer` (1초 간격) + `QTime.secsTo()` 방식
- "경과 시간: MM:SS" 형식으로 프로그레스바 옆에 표시
- 로딩 완료 시 타이머 정지

### R3: 행정경계 4분리

- `sgis_hjd` 테이블에서 `LENGTH(adm_cd)` 필터로 레벨 분리
  - 시도: LENGTH=2 / 시군구: LENGTH=5 / 읍면동: LENGTH=8 / 법정리: LENGTH=10
- 라벨: QGIS expression `regexp_substr("adm_nm", '[^ ]+$')` → 마지막 단어만 표시
- 스타일: 시도/시군구 → `경계_시군구`, 읍면동 → `경계_읍면동`, 법정리 → `경계_리`

### R4: 하천 라벨

- `_on_task_finished()`에서 하천경계 로드 완료 후 `_apply_river_labels()` 호출
- 필드 후보: `하천명` → `소하천명` → `RIVER_NM` → `river_nm` (우선순위 탐색)
- 필드 없으면 `Qgis.Warning` 로그 후 스킵 (에러 없음)

### R10: 드래그앤드롭 (Step 5)

- `DropZoneWidget(QFrame)`: `setAcceptDrops(True)`
- 지원 형식: `.shp`, `.gpkg`, `.geojson`
- 파일 URL + `application/qgis.layertreemodeldata` MIME 타입 지원
- 기존 파일 선택 다이얼로그도 유지 (폴백)

### R11: 인코딩 처리

- `layer.setProviderEncoding(encoding)` — 레이어 생성 직후, 클립핑 전에 적용
- 자동감지: EUC-KR → CP949 → UTF-8 순서로 try/except
- `chardet` 미사용 (QGIS 표준 Python에 없음)

### R13: 계획시설 템플릿 스타일 (Step 7)

- 9색: 빨강, 노랑, 초록, 하늘, 파랑, 선홍, 검정, 회색, 주황(#FF8000)
- 2타입: 이중점선, 이중실선
- `QGridLayout`으로 18개 셀 표시 (색상 미리보기 + 선택)
- 선택한 스타일을 `StyleManager`에서 직접 XML 심볼로 적용
